EigenCalc

Cálculo de Autovalores e Autovetores em C

C Álgebra Linear Método da Potência CSV Export Análise Numérica

Sobre o Projeto

O EigenCalc é uma aplicação em C que calcula autovalores e autovetores de matrizes 3x3 usando o método da potência com precisão numérica avançada.

Características Principais:

  • Implementação do método da potência iterativo
  • Precisão de até 1e-10
  • Exportação de resultados em formato CSV
  • Verificação numérica dos resultados
  • Interface de linha de comando intuitiva
Matemática Computacional Algoritmos Iterativos Processamento de Matrizes

Código Fonte

Implementação completa do algoritmo de cálculo de autovalores:

eigencalc.c 
#include <stdio.h>
#include <stdlib.h>
#include <math.h>
#include <string.h>

#define MAX_ITER 1000
#define TOLERANCE 1e-10
#define SIZE 3

// Função para multiplicar matriz por vetor
void matriz_vetor_mult(double matriz[SIZE][SIZE], double vetor[SIZE], double resultado[SIZE]) {
    for (int i = 0; i < SIZE; i++) {
        resultado[i] = 0.0;
        for (int j = 0; j < SIZE; j++) {
            resultado[i] += matriz[i][j] * vetor[j];
        }
    }
}

// Função para calcular o produto escalar
double produto_escalar(double v1[SIZE], double v2[SIZE]) {
    double resultado = 0.0;
    for (int i = 0; i < SIZE; i++) {
        resultado += v1[i] * v2[i];
    }
    return resultado;
}

// Função para normalizar um vetor
void normalizar_vetor(double vetor[SIZE]) {
    double norma = sqrt(produto_escalar(vetor, vetor));
    if (norma > 0) {
        for (int i = 0; i < SIZE; i++) {
            vetor[i] /= norma;
        }
    }
}

// Método da potência para encontrar autovalor e autovetor
void metodo_potencia(double matriz[SIZE][SIZE], double *autovalor, double autovetor[SIZE]) {
    double vetor_anterior[SIZE], vetor_atual[SIZE];
    double autovalor_anterior;
    int iter;
    
    // Vetor inicial
    for (int i = 0; i < SIZE; i++) {
        vetor_atual[i] = 1.0;
    }
    normalizar_vetor(vetor_atual);
    
    for (iter = 0; iter < MAX_ITER; iter++) {
        memcpy(vetor_anterior, vetor_atual, sizeof(vetor_anterior));
        autovalor_anterior = *autovalor;
        
        matriz_vetor_mult(matriz, vetor_anterior, vetor_atual);
        *autovalor = produto_escalar(vetor_anterior, vetor_atual);
        normalizar_vetor(vetor_atual);
        
        if (fabs(*autovalor - autovalor_anterior) < TOLERANCE) {
            break;
        }
    }
    
    memcpy(autovetor, vetor_atual, sizeof(vetor_atual));
    printf("Convergiu em %d iterações\n", iter + 1);
}
